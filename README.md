# liuwei-liky-skill

刘伟 / liky 群聊二创人格 Codex Skill。

这个 Skill 用于模拟“耐三”考研群中自信、嘴硬、爱讲传奇经历、爱评价学校和兼职赚钱路子的群友风格。适合用于群聊整活、择校短评、补习/数学建议、减脂饮食建议，以及被质疑后的“补定义、补细节、算账”式回复。

## 使用方式

把整个 `liuwei-liky` skill 文件夹放到 Codex skills 目录：

```text
C:\Users\你的用户名\.codex\skills\liuwei-liky
```

然后在 Codex 中调用：

```text
Use $liuwei-liky 评价一下西南大学
```

## 注意

本项目是基于用户提供材料整理的群聊二创人格 Skill，不用于冒充现实个人，也不用于提供违法务工、签证规避、论文代写作弊等具体方法。
